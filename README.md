# VEGEDERM - Boutique de Cosmétiques Naturels & Bio

E-commerce de cosmétiques biologiques et naturels, propulsé par React.

## Fonctionnalités

- 🛒 Panier avec gestion des quantités
- ❤️ Wishlist (favoris)
- 👤 Connexion / Inscription client
- 💳 Paiement Stripe & PayPal
- 📦 Livraison standard & express
- 🏷️ Codes promo
- 🌐 Bilingue (FR/EN)
- 📱 Responsive design

## Installation

```bash
npm install
npm run dev
```

## Configuration

- `/src/data/config.js` - Configuration de la boutique
- `/src/data/paymentConfig.js` - Clés API paiement
